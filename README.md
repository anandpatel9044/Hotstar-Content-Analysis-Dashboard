# 📊 Hotstar Content Analysis Dashboard

An interactive Excel dashboard built to analyze streaming content data and uncover insights related to genres, audience targeting, and content trends.

---

## 🚀 Project Overview

This project focuses on transforming raw Hotstar content data into meaningful insights using Microsoft Excel. The dashboard enables quick exploration of content distribution, trends over time, and viewer-oriented patterns.

---

## 🎯 Objectives

- Analyze content distribution across genres  
- Identify trends in content release over the years  
- Understand audience targeting through age ratings  
- Evaluate content depth using seasons and episodes  
- Build an interactive and visually appealing dashboard  

---

## 🛠️ Tools & Techniques

- **Microsoft Excel**
  - Pivot Tables  
  - Charts (Bar, Line, Pie)  
  - Slicers (Interactive Filters)  
- Data Cleaning & Preparation  
- Dashboard Design  

---

## 📊 Dashboard Features

- 🔹 KPI Cards  
  - Total Titles  
  - Total Movies  
  - Total TV Shows  

- 🔹 Genre Distribution  
- 🔹 Year-wise Trend  
- 🔹 Age Rating Analysis  
- 🔹 Runtime Analysis  
- 🔹 TV Show Depth Analysis  

- 🔹 Interactive Filters (Slicers)  
  - Genre  
  - Year  
  - Age Rating  

---

## 📈 Key Insights

- Drama and Reality are the most dominant genres  
- Reality shows have higher episode counts (binge-friendly content)  
- Content production has increased significantly over time  
- Most content targets a broad audience (U/A category)
